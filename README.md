# JDBC Role-Based Authentication System

A secure console-based login system built using Core Java and JDBC.
Includes password hashing, role-based access control, and account lock mechanism.

## Features
- User registration & login
- SHA-256 password hashing
- Admin & User roles
- Account lock after failed attempts
- JDBC with MySQL/PostgreSQL

## Technologies
- Java
- JDBC
- MySQL / PostgreSQL
- Git
